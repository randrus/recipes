# Tapenade

## Ingredients
* olives - mix of pitted kalamata / green sicilian
* capers
* garlic
* olive oil

## Preparation
* put olives && capers into blender
* buzz == blenderize 
* add olive oil for consistency 

## Have tried:
* small pickles (cornichons)
